# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2026-01-24

### Added
- Initial release
- Search packages from pub.dev API
- Add packages to Flutter projects with one click
- Automatic Flutter SDK detection
- Manual Flutter SDK selection
- Project folder selection
- Package details view (version, description, homepage)
- Light and dark theme support (follows system theme)
- Modern Material Design 3 interface
- Cross-platform support (Windows, macOS, Linux)
- Error handling with user-friendly messages
- Debounced search for better performance
